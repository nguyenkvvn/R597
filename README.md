# R597
Printable receiver for the Remington 597

by nguyenkvvn

## Description
A printable receiver for the Remington 597 .22 LR part kits. Not compatible with .17 HMR part kits.

## Variants
The following variants are included:

- **Standard** - Standard, slick-top receiver with no rails
- **Picatinny** - Receiver with picatinny rails
